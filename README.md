# Sign Language Recognition System

A real-time Sign Language Recognition System built using Python, OpenCV, MediaPipe, and TensorFlow/Keras. The project detects hand gestures through a webcam and recognizes sign language alphabets using a trained deep learning model.

## Features

- Real-time hand detection
- Sign language alphabet recognition
- MediaPipe hand landmark detection
- Deep Learning model using TensorFlow/Keras
- Webcam-based live prediction
- Data collection and model training scripts

## Technologies Used

- Python
- OpenCV
- MediaPipe
- TensorFlow / Keras
- NumPy
- Jupyter Notebook
